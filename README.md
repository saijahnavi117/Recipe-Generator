A web application that suggests Indian recipes based on the ingredients you have. Easily filter recipes by cuisine, course, diet type, and servings.

---

## Features

- Suggest recipes based on available ingredients.
- Filter recipes by:
  - **Cuisine**: North Indian, South Indian, etc.
  - **Course**: Breakfast, Lunch, Dinner, Snacks
  - **Diet type**: Vegetarian, Non-Vegetarian, Vegan
  - **Servings**
- User-friendly interface built with **HTML, CSS, JavaScript**.
- Backend powered by **Python & Flask**.
- Recipe management and data handling using **Pandas**.
