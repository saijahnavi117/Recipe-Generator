# Indian Recipe Generator üçõ

A **Flask-based web application** that recommends Indian recipes based on user-input ingredients and preferences. The app allows filtering by cuisine, course, diet type, and servings for accurate recipe matching.

---

## Features

- Suggests recipes based on the ingredients you have.
- Filter recipes by:
  - Cuisine (North Indian, South Indian, etc.)
  - Course (Breakfast, Lunch, Dinner, Snacks)
  - Diet type (Vegetarian, Non-Vegetarian, Vegan)
  - Servings
- User-friendly interface built with **HTML, CSS, JavaScript**.
- Backend powered by **Python & Flask**.
- Data handling with **Pandas** for recipe management.

---


## Installation

Installation

Clone the repository:

git clone https://github.com/yourusername/Indian-Recipe-Generator.git


Navigate into the backend folder:

cd Indian-Recipe-Generator/backend


Install required Python packages:

pip install -r requirements.txt


Run the Flask app:

python app.py


Open your browser and go to:

http://127.0.0.1:5000/

Tech Stack

Backend: Python, Flask

Frontend: HTML, CSS, JavaScript

Data Handling: Pandas

Deployment (Optional): Heroku / Render / PythonAnywhere
